# **Risk en Java**
Este proyecto es una implementación del clásico juego de estrategia *Risk*.
Este repositorio es un trabajo para la asignatura de Sistemas Distribuidos, por lo que los objetivos del desarrollo han sido los siguientes:

- Programación y gestión de hilos concurrentes.
- Uso de Sockets para la comunicación entre los jugadores.

El gameplay es una parte superficial del proyecto, el principal interes del desarrollo reside en el sistema de salas con un servidor central y el sistema de turnos P2P.


### Requisitos previos
- [JavaFX 17+](https://openjfx.io/)

### Uso

La clase Servidor ha de iniciarse primero, es la encargada de coordinar sistema de salas.

La clase App contiene la interfaz de usuario.
